# Kairo — Round 3: 20-Phase 10× Program (2026-07-23)

**Goal:** make the reminder/presence story real on BOTH platforms (the #1 ADHD
need — time-based nudges), bring iOS to full screen-parity with web, add
genuinely new ADHD capabilities, and harden offline/perf/a11y — all verified.

Grounding (verified 2026-07-23):
- Web is mature + PWA scaffolded (`public/sw.js`, `manifest.json`), notification
  service exists (`computeNotificationJobs`, `pushSubscriptions` table) but SW
  push handlers + web-push delivery are **not wired** ("full 3B" TODO).
- **iOS has NO local notifications at all** — biggest gap for a planner.
- iOS is missing web screens: **Templates, Month view, Search/Command**.
- Calendar is ICS export only (read feed), not two-way.
- Both platforms have garden/reflection/rituals/peak nudge/soundscape.

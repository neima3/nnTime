# Kairo (nnTime)

**Time you can see.** A visual daily planner for brains that plan differently —
ADHD/neurodivergent-first, delightful for everyone. Web (desktop + mobile) now,
native iOS next. Live at [time.neima.me](https://time.neima.me).

- Plan: `docs/plans/2026-07-12-kairo-roadmap.md`
- Design contract: `docs/design/design-spec.md`
- Agent handoff: `docs/plans/kairo-agent-prompt.md`
